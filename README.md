# 项目简介

## 核心功能
- 基于 YOLOv8 的电梯电动车检测系统
- 支持 6 类目标检测：['person', 'Electric-bicycle', 'bicycle', 'Baby-carriage', 'wheelchair', 'YES']
- 支持多种输入源检测：单张图片、图片文件夹、视频文件、本地摄像头
- 实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并查看所有结果
- 支持检测结果保存
- 可自定义置信度和 IOU 阈值
- 可选择是否显示标签和原图
- 可选择不同检测模型

## 适用场景
- 电梯安全管理，防止电动车进入电梯
- 公共场所安全监控
- 人员与车辆识别统计
- 安防监控系统

## 技术特点
- 使用 Python 3.9 开发
- 集成 OpenCV-Python、PyQt5、Ultralytics、Torch1.9 等技术栈
- 包含 4200 张标注数据集
- 提供完整源码和训练好的模型
- 配置成功环境后可直接使用
